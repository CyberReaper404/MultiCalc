🧮 Calculadora Científica - C# e JavaScript

Este repositório reúne duas versões de uma calculadora científica desenvolvidas como parte de um projeto maior. Inicialmente, fiz a versão em Python (disponível em outro repositório) e, depois, tive a ideia de implementar a mesma calculadora em C# e JavaScript para adquirir mais conhecimento e praticar diferentes linguagens.

Ambas as versões rodam no terminal com menus interativos e as mesmas funcionalidades.

---

📅 Sobre este projeto

Os códigos foram desenvolvidos originalmente em 2021 como parte de um trabalho acadêmico. O objetivo era implementar a mesma solução em diferentes linguagens para comparar sintaxes, estruturas de controle e abordagens.

Primeiro fiz a versão em Python (repositório: CalculadoraCientifica). Depois, para expandir meus conhecimentos, decidi refazer o projeto em C# e JavaScript, que estão reunidos aqui.

Na época eu ainda não conhecia o Git e não sabia como versionar ou compartilhar projetos. Agora estou organizando e publicando meus projetos antigos para documentar minha evolução como desenvolvedora.

⚙️ Funcionalidades (em ambas as versões)

Operações básicas: Adição, Subtração, Multiplicação, Divisão
Potência e raízes: Potenciação, Raiz quadrada, Raiz cúbica
Trigonometria: Seno, Cosseno, Tangente, Arco seno, Arco cosseno, Arco tangente (ângulos em graus)
Logaritmos: Logaritmo natural (ln), Logaritmo base 10, Logaritmo em qualquer base
Números complexos: Adição, Multiplicação, Módulo, Forma polar

Todas as operações possuem tratamento de erros (divisão por zero, logaritmo de número negativo, domínio das funções inversas, etc.)

🚀 Como executar

Versão C#

Pré-requisitos:
- .NET SDK (versão 6.0 ou superior) - https://dotnet.microsoft.com/download

Executar:
cd csharp
csc Calculadora.cs
Calculadora.exe

Versão JavaScript

Pré-requisitos:
- Node.js (versão 12 ou superior) - https://nodejs.org/

Executar:
cd javascript
node calculadora.js

🎯 Por que esses projetos?

Esses foram meus primeiros contatos mais sérios com C# e JavaScript durante a faculdade. O objetivo era praticar:

- Estruturas de repetição (do-while, for)
- Condicionais (if/else, switch)
- Funções/métodos e organização de código
- Entrada e saída no console
- Uso de bibliotecas matemáticas
- Lógica de programação em diferentes paradigmas

São projetos simples, mas que representam o início da minha jornada com essas tecnologias.
