# MultiCalc
🧮 Duas linguagens, uma calculadora!  Versões em C# e JavaScript da calculadora científica que fiz na faculdade em 2021. Tudo no terminal, menus interativos, e as mesmas funcionalidades nos dois.  🔹 C# → /csharp 🔹 JS → /javascript  Operações básicas, potência, trigonometria, logaritmos e números complexos.
